# Пользовательское соглашение — OfficeDesk Конвертер

**Дата вступления в силу:** _подставить_

## 1. Общие положения

Настоящее Соглашение регулирует использование приложения «OfficeDesk Конвертер» (далее — «Приложение»). Устанавливая или используя Приложение, вы соглашаетесь с условиями ниже.

## 2. Назначение

Приложение предназначено для конвертации документов между форматами на устройстве пользователя. Приложение **не предназначено** для:
- получения доступа к чужим документам;
- обхода защиты документов (паролей, шифрования, DRM);
- массовой обработки документов в коммерческих целях без соблюдения авторских прав на их содержимое.

## 3. Ответственность пользователя

Пользователь обязуется:
- использовать Приложение только для конвертации документов, которыми он владеет или на которые имеет права;
- не использовать Приложение для нарушения авторских прав третьих лиц;
- не загружать в Приложение материалы, запрещённые законодательством Российской Федерации.

## 4. Ограничение ответственности

Приложение предоставляется «как есть». Разработчик не несёт ответственности за:
- потерю или повреждение исходных файлов в процессе конвертации (рекомендуем хранить резервные копии);
- неточности форматирования при конвертации сложных документов;
- последствия использования сконвертированных файлов.

## 5. Конфиденциальность

Обработка файлов происходит локально, на устройстве пользователя. Подробности — в Политике конфиденциальности.

## 6. Связь с другими приложениями

Приложение может предлагать открыть результат конвертации в Приложении «OfficeDesk Читалка». Установка и использование Читалки — на усмотрение пользователя.

## 7. Изменение и прекращение

Разработчик вправе изменять Соглашение. Актуальная редакция публикуется по адресу политики/соглашения. Использование Приложения после изменений означает согласие с новой редакцией.
