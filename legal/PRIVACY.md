# Политика конфиденциальности — OfficeDesk Конвертер

**Дата вступления в силу:** _подставить_

Приложение «OfficeDesk Конвертер» (далее — «Приложение») разработано для локальной конвертации документов на устройстве пользователя.

## 1. Какие данные мы собираем

**Никакие.**

Приложение не собирает, не передаёт и не хранит на удалённых серверах:
- персональные данные;
- содержимое файлов пользователя;
- метаданные файлов (имена, размеры);
- идентификаторы устройства, рекламные идентификаторы;
- данные о местоположении;
- историю действий пользователя.

## 2. Где обрабатываются файлы

Все файлы, которые пользователь выбирает для конвертации, обрабатываются **исключительно на устройстве пользователя**. Файлы не отправляются на серверы Приложения или третьих лиц.

## 3. Какие разрешения запрашивает Приложение

Приложение запрашивает доступ к выбранному пользователем файлу через системный диалог Android (Storage Access Framework). Это означает, что Приложение получает доступ только к тем файлам, которые пользователь выбрал явно. Приложение **не получает** доступ ко всему хранилищу.

## 4. Хранение результатов

Сконвертированные файлы сохраняются в приватной директории Приложения на устройстве пользователя. При удалении Приложения эти файлы автоматически удаляются.

## 5. Сетевой доступ

Приложение **не использует интернет** для обработки документов. Сетевые подключения могут происходить только при переходе пользователя по ссылкам в разделе «О программе» (открытие политики, соглашения, почтового клиента) или при открытии страницы Читалки в RuStore.

## 6. Сторонние сервисы

Приложение не использует сторонние SDK для аналитики, рекламы или сбора данных.

## 7. Изменения в политике

При изменении политики обновлённая версия будет опубликована по этому адресу. Дата изменения указывается вверху документа.

## 8. Контакты

По вопросам, связанным с обработкой данных, обращайтесь:
**support@officedesk.ru**
